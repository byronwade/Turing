# Compatibility and Conformance Gates

Status: release-gate framework  
Owner: web platform, JavaScript, networking, quality, DevTools, and product

Compatibility evidence includes pinned WPT, Test262, WebDriver and WebDriver BiDi revisions, controlled application corpora, protocol conformance, accessibility behavior, and explicit proprietary gaps.

The denominator includes failures, crashes, timeouts, unsupported tests, and platform exclusions. Site-specific interventions are public, owned, tested, and expiring.

WebDriver BiDi is tracked by exact Working Draft or editor revision because its specification remains in active development. Turing does not promise a floating “latest” protocol contract.
