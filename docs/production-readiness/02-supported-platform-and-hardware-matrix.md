# Supported Platform and Hardware Matrix

Status: no production platform selected  
Owner: platform, release, accessibility, performance, and support

A platform is supported only when installation, signing, sandboxing, updates, rollback, text, IME, accessibility, GPU/software rendering, power lifecycle, crash reporting, and support procedures pass release-equivalent tests.

The matrix records operating-system versions, CPU architectures, GPU requirements, software fallback, assistive technologies, package formats, minimum hardware tiers, unsupported configurations, and end dates.

Experimental compilation is not support. Community reports on an unowned platform do not create a security-patch obligation unless the support matrix is deliberately changed.
