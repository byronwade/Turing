# Legal, Signing, and Human Release Authority

Status: mandatory human-gate baseline  
Owner: legal, release, security, program, and repository owner

Before stable distribution, qualified humans approve licensing, contribution terms, trademarks, patents/codecs, DRM, assets, privacy, telemetry, AI-provider terms, export controls, sanctions, app-store obligations, accessibility obligations, automatic updates, vulnerability safe harbor, and support commitments.

Offline root keys and production signing keys are separated from coding agents and ordinary CI. Key ceremonies, threshold policy, hardware protection, rotation, revocation, audit, backup, and compromise recovery are documented and rehearsed.

Only designated human release authority may approve stable promotion. Automation verifies evidence and performs mechanical steps but cannot manufacture legal, security, or support approval.
