# Stable v1 Scope and Non-scope

Status: unapproved scope framework  
Owner: product, program, architecture, support, and release

Stable v1 must name a finite capability set, not “everything Chrome supports.” The contract records supported browser workflows, web-platform areas, profile formats, Plug-in API, embedding API, platforms, languages, accessibility technologies, release duration, and proprietary gaps.

Every supported item has acceptance evidence and an owner. Every excluded item is discoverable and cannot be implied by user-agent branding or marketing.

Likely early exclusions include mobile, proprietary DRM, vendor account services, broad enterprise management, unrestricted third-party native Plug-ins, consequential agent actions, and unverified hardware/device APIs.

The machine scope remains `unapproved`. Promotion requires compatibility, security, accessibility, update, recovery, operations, staffing, and legal evidence.
